# Suggested Commands
- `npm run dev` – start the Next.js development server (Turbopack).
- `npm run build` – create a production build.
- `npm run start` – serve the production build.
- `npm run lint` – run ESLint checks (Next core web vitals + TypeScript rules).
- `npm run generate-types` – regenerate Supabase TypeScript definitions into `types/database.ts`.
